# Ketube Client

```bash
## Setup
# Use Node.js Version 18.19.0
- brew install node@18
- brew link node@18

# ถ้าใช้ Node Version อื่นอยู่ให้
- brew unlink node

# เข้ามาใน Project
- npm install

# Start Project Develop Mode
- npm run dev

# Start Project SSR Mode
- npm run build
- npm run preview

# ขั้นตอนก่อน Commit Code *ทำก่อนทุกครั้ง
- npm run lintfix
```
